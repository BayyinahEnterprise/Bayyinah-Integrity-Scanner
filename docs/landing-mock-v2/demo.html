§ Demo·Document Firewall

The page held up to the light, before the model reads it.

Upload a PDF. The scanner runs the same 159 mechanisms that power bayyinah.dev. If the firewall verdict is sahih (clean), Anthropic's Sonnet 4.6 summarizes the document. If it is munafiq (concealed) or mughlaq (locked), the model never sees the file, and zero LLM tokens are consumed.

The argument is one layer below the model. Every benchmark in the JailbreakBench / AgentHarm / RobustBench lineage assumes the adversarial input has already reached the LLM. The firewall argues for verification before ingestion: deterministic, no inference, no optimizer, no surface for prompt injection because the gate has no goal.
