Bayyinah Demo: Document Firewall for AI Pipelines

Upload a PDF. If it conceals anything (Tier 1 finding, scan incomplete, or high-confidence Tier 2), Claude never sees it. If clean, the document is summarized and the token count is reported. Zero LLM tokens consumed on a blocked document.

Stateless: no file is persisted, no request body is logged. The demo runs the same scanner that powers bayyinah.dev.
